# Course Media Guidelines

This document defines recommended sizes and ratios for course media assets.
Use these as targets to keep visuals consistent across the app.

## Banner (Course Detail Header)
- Purpose: large visual at the top of the course detail page.
- Recommended size: **1200 x 600 px** (2:1).
- Minimum size: **900 x 450 px**.
- Safe area: keep key content within the central **80%** width and **60%** height.

## Course Cover Image (Course Card / Summary)
- Purpose: small square image used in cards and summary blocks.
- Recommended size: **800 x 800 px** (1:1).
- Minimum size: **400 x 400 px**.
- Safe area: keep key content within the central **70%**.

## Screenshots (Course Screens Carousel)
- Purpose: horizontal gallery of in‑course screens.
- Recommended size: **1280 x 720 px** (16:9).
- Minimum size: **960 x 540 px**.
- Safe area: keep key content within the central **85%** width and **75%** height.

## File Format & Quality
- Prefer **JPEG** for photos, **PNG** for UI screenshots.
- Compress for web/mobile: target **150–300 KB** per image when possible.
- Avoid heavy text near edges (it may be cropped on different devices).
