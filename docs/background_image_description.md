# Background Image Description (Primary + White)

## Purpose
A calm, readable background image that complements the app's primary green and white UI elements.

## Visual Style
- Minimalistic, modern, soft gradients
- Smooth flowing waves with gentle depth
- Clean, airy feel to keep UI text readable

## Composition
- Wide, landscape orientation (usable as full-screen background)
- Soft wave bands flowing diagonally or horizontally
- Waves should be subtle, not busy, leaving large calm areas for content
- Optional faint light bloom in one corner for depth

## Resolution & Format
- Primary: 4096×2304 (16:9) PNG
- Secondary: 2880×1800 (16:10) PNG
- Optional export: 1920×1080 JPG (quality 90) for lightweight previews

## Color Palette (suggested)
- Deep charcoal background base (near black, not pure)
- Soft gray wave highlights
- Subtle cool-neutral tones to avoid warmth
- Tiny hints of desaturated green glow to harmonize with primary

## Example Palette (HEX)
- Base: #0F1113
- Wave shadow: #1A1D20
- Mid: #23272B
- Highlight: #2E3338
- Accent glow: #2A3A34 (very subtle)

## Lighting
- Diffuse, even lighting
- Low contrast so text remains legible
- Smooth transitions between tones

## Texture
- No noise or grain
- No sharp edges
- Soft, silky gradients

## Notes
- Avoid high saturation colors
- Avoid strong patterns or busy shapes
- The image should sit behind glass UI and not compete with content
